04 / Components

UI component library

Every component is built from the same tokens. No one-offs. Hover states, border treatments, and accent usage follow the same rules everywhere.

Buttons

Primary — Filled accent

Background: --color-accent. Text: near-black #0c0b09. Hover: translateY(-1px) + darken. Font: DM Mono, uppercase, tracked.

Ghost — Transparent border

Border: --color-border-mid. Hover: border-strong + text-primary. Never use accent fill on ghost.

Border glow — animated beam

Glow Primary — persistent rotating beam

Conic beam spins 360° every 4s at inset[-300%], visible only through the 1.5px gap between outer edge and inner cover. Bottom radial glow expands on hover. Use for primary hero CTA only — never repeat on the same page.

Ghost Glow — beam appears on hover

At rest: muted border, no animation (no GPU cost). On hover: conic beam fades in at 0.3s, border fades out, bottom radial halo appears. Secondary CTA only — pairs with the persistent glow primary above.

Motion principles

All animation is intentional. Nothing loops for decoration. Motion either reveals content, confirms an interaction, or signals system state. Easing is always ease-out — fast start, graceful end.

Each word = its own wrapper/content pair. 80ms stagger per word. Italic accent on last word — the payoff.

How the mask works

.text-reveal-wrapper
overflow: hidden — acts as the clipping mask. The child is invisible below this boundary.
.text-reveal-content
Starts at translateY(110%) — fully hidden. On .reveal-active: slides to translateY(0). Duration 1.1s ease-out.
Trigger
IntersectionObserver adds .reveal-active to the wrapper parent when it enters viewport at threshold 0.15.

Use for: hero headlines, section titles, CTA banners. Never use on body copy — reserve for display text only. In Framer: use scroll-triggered opacity + y-position on a clipped container.

07 / Brand Voice

How Vellumwire speaks

Every word on the site is a brand decision. Direct, specific, zero throat-clearing. The copy earns trust the same way the design does — by being precise, not by being warm.

The brand IS →
→ Direct. Names the problem without softening it.
→ Specific. Describes exact failure modes, not vague "challenges."
→ Confident. States the diagnosis. Earns the prescription.
→ Dry wit, briefly. One sharp line per section, not a tone.
→ Results-framed. Every feature described by its outcome.
"A beautiful website that doesn't generate calls is a liability."
"Visitors can't tell what to do next — so they bounce, or call a competitor."
The brand is NOT ×
× Agency buzzwords: "holistic," "ecosystem," "leverage," "synergy."
× Warm and vague: "We care deeply about your success." Everyone says this.
× Over-qualified: "We try to help businesses potentially improve conversion."
× Jargon-first: "UX/UI, CRO, GA4 funnels" as your opening line.
× Humble to a fault: Don't apologise for having a strong point of view.
"We provide comprehensive web design solutions tailored to help service businesses optimise their digital presence and improve user engagement metrics."
