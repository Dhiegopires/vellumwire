Mapeamento de Intenção

As 5 páginas de maior prioridade: home, contact, packages, insights, work
Home/index.html

Intenção

Comercial. Visitante já sabe que tem um problema de conversão e está avaliando se vale falar com a Vellumwire.

Persona / mindset

Dono ou responsável de marketing de um negócio de serviço, chegando via busca direta ou indicação, comparando 2-3 agências.

Estágio de funil

Topo a meio

Seções obrigatórias

  • Hero com promessa + CTA único
  • Diagnóstico do problema ("The Leak")
  • Prova de resultado (número Fiter — ver Step 1)
  • Serviços com CTA segmentado
  • FAQ de objeção

Links internos

Envia para: work, packages, contact (todos os CTAs de serviço). Recebe de: nenhuma — é a porta de entrada principal.

Contact/contact.html

Intenção

Transacional. Visitante já decidiu agir, está convertendo.

Persona / mindset

Mesmo persona da home, agora pronto para preencher formulário, com a query ?service=X indicando de qual card ele veio.

Estágio de funil

Fundo

Seções obrigatórias

  • Formulário com proteção Turnstile
  • Confirmação visual do serviço selecionado (via query param)
  • Sem distração nem CTA secundário

Links internos

Recebe de: home, packages, studio, work, todos os case studies. Não envia para nenhuma página (página terminal do funil).

Packages/packages.html

Intenção

Comercial, com sinal de preço explícito na busca.

Persona / mindset

Visitante comparando custo entre agências antes de marcar call — já passou a fase de "o que é isso" e está em "quanto custa".

Estágio de funil

Meio a fundo

Seções obrigatórias

  • 5 tiers de projeto único ($350 a $4.500-$8.000)
  • Seção "After launch" com retainer mensal Hold ($400) / Build ($800)
  • CTA por tier

Links internos

Recebe de: nav global. Envia para: contact (por tier).

Correção

Releitura completa de packages.html mostra que não há inconsistência: a página tem 5 tiers de projeto único ($350 Diagnóstico até $4.500-$8.000 Marca+Site), idênticos à faixa citada no FAQ da home, mais uma seção separada "After launch" com retainer mensal Hold ($400) / Build ($800) para manutenção pós-lançamento. São duas ofertas complementares, não concorrentes — a primeira leitura deste documento havia comparado as duas seções como se fossem a mesma oferta, o que estava errado.

Insights/insights.html

Intenção

Informacional, topo de funil — quem busca isso ainda não sabe que precisa de uma agência.

Persona / mindset

Pesquisando o próprio problema antes de procurar solução ("meu site não converte, por quê?").

Estágio de funil

Topo

Seções obrigatórias

  • Lista de artigos (hoje vazia)
  • CTA suave para diagnóstico ao final de cada artigo

Links internos

Recebe de: nav global. Deveria enviar para: contact e para páginas de serviço relacionadas ao tema do artigo — hoje não envia para nada porque não há conteúdo.

Work/work.html

Intenção

Comercial tardio — validação de prova antes de decidir.

Persona / mindset

Já leu a home, quer ver se a Vellumwire realmente entrega o que promete antes de preencher o formulário.

Estágio de funil

Meio a fundo

Seções obrigatórias

  • Filtro de projetos
  • Cards com resultado resumido por case
  • Link para case completo

Links internos

Recebe de: nav global, home. Envia para: os 3 case studies, contact.
